== Description ==
The GPA Calculator plugin allows users to easily calculate their GPA by adding courses, grades, and credits. It supports both weighted and unweighted GPA calculations.

== Installation ==
1. Upload the `gpa-calculator` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the GPA Calculator page from the WordPress dashboard to start using the plugin.

== Frequently Asked Questions ==

= How do I add a new semester? =
Click on the "+ Add Semester" button, then enter the semester title and add your courses.

= How do I add a new course? =
Under the respective semester, click on the "+ Add Course" button. Fill in the course name, grade, credits, and course type.

= What grades can I input? =
You can input standard letter grades such as A+, A, B+, etc.

= How is the GPA calculated? =
The plugin calculates both weighted and unweighted GPAs based on the courses and credits you input.
